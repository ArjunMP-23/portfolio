# Arjun M P Portfolio

A personal portfolio website showcasing my skills, education, and featured projects.

## Features
- Responsive design with light/dark mode toggle
- Animated project cards
- Technical skills displayed in modern card layout
- Education and summary sections

## Tech Stack
- HTML, CSS, JavaScript
- Modern CSS (Flexbox, Grid, Animations)

## Usage
1. Clone the repository
2. Open `index.html` in your browser

## Sections
- **Summary**: Brief introduction and contact info
- **Education**: Academic background
- **Projects**: Featured web development projects
- **Skills**: Technical and soft skills

## Customization
Feel free to edit the content in `index.html` and styles in `style.css` to personalize your portfolio.
